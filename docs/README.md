# Scenario Tool Documentation

> A comprehensive guide to using the Linea Analytics **Scenario Tool** for marketing budget optimisation.

## Overview

The [**Scenario Tool**](https://app.linea-analytics.com/) helps marketing teams optimise their media budget allocation across channels and campaigns. It uses advanced analytics and Marketing Mix Modeling (MMM) to provide data-driven recommendations.

## Features

- Run different types of optimisation scenarios:
  - Fixed budget allocation
  - Target uplift calculations
  - Unconstrained optimisation
- Set custom time periods from monthly to annual
- Apply what-if factors to model future changes
- Compare multiple scenarios side by side
- Incorporate channel synergies and constraints
