# Linea Client Portal Documentation

> Your guide to the **Linea Client Portal** — covering data onboarding, API authentication, and the Scenario Tool for marketing budget optimisation.

## Overview

The [**Linea Client Portal**](https://app.linea-analytics.com/) gives marketing teams everything they need to go from raw data to actionable insights. These docs cover two core areas:

- **MMM Onboarding** — how we connect your data sources, authenticate APIs, and get your MMM model up and running.
- **The Scenario Tool** — how to use the portal to optimise your media budget, run what-if scenarios, and plan your channel mix.
