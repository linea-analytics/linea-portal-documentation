# Scenario API

The functionality of the Scenario Tool are also available as an API. This will allow you to integrate the speed and flexibility of the **Scenario Tool** directly into your existing processes. 

## Modelling

Speed up your current modelling capabilities by calling the **Scenario API** directly from your own code base and/or modelling tools. 

## Planning

Enhance your planning tools by integrating **Scenario API** in any of your existing platforms:
- Cloud based
- Spread sheets 
- Bespoke software
